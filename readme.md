# 📚 Cohort Learning Notes

This repository contains my **learning notes and code explanations** created while studying different topics during my cohort sessions.

Instead of just writing theoretical notes, I follow a **“teach while learning” approach**.
Whenever I learn a concept, I write code and explain it through comments as if I am teaching someone else.

This helps me:

* Understand concepts more deeply
* Organize my thoughts clearly
* Convert these explanations into blog posts later

---

# 🎯 Purpose of this Repository

This repo acts as my **structured learning workspace**.

The workflow is simple:

1. Learn a concept in the cohort
2. Write code examples
3. Add detailed comments explaining the logic
4. Push the notes to this repository
5. Convert the explanation into a **Medium blog post**

So this repository is essentially the **source material for my blogs**.

---

# 🧠 Learning Method

Instead of passive notes, I write explanations in code


This style helps simulate **teaching someone**, which reinforces understanding.
